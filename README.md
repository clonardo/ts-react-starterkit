# TS React StarterKit

Boilerplate to setup a web project with Webpack 4, React 16, Typescript 3, CSS Modules, Node SASS or PostCSS Plugin ecosystem.


## Requirements

- Node >= v8.5.x
- Yarn >= v1.5.x


## Available tasks

```sh
# Runs development server (Webpack Dev Server)
$ yarn start

# Build command
$ yarn build

# Lint with Stylelint and TSLint
$ yarn lint
```
