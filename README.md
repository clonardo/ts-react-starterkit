## Requirements

- Node >= 8.5.x
- Npm >= 6.x
- Yarn >= 1.5.x


## Available tasks

```sh
# Runs development server (Webpack Dev Server)
$ yarn start

# Build command
$ yarn build

# Lint with Stylelint and TSLint
$ yarn lint
```
